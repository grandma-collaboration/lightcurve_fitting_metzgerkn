# Real LSST-like Inputs

These files are included to show the real input schema used during the internship
analysis. The full feature table is not copied here; instead,
`features_kn_cut_1000_real_excerpt.csv` keeps only the rows matching the copied
light curves.

Copied light curves:

- `lightcurve_LSSTlike_0090.csv`
- `lightcurve_LSSTlike_0495.csv`
- `lightcurve_LSSTlike_0637.csv`

These use the Alex/LSST-like columns:

```text
observationStartMJD,band,m5,t_days,fiesta_filter,mag_true,snr_exp,mag_err,snr_obs,detected,mag_obs,mag_ulim
```
